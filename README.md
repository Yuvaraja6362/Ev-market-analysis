
## Project Title

EV Market Analysis Dashboard using Power BI

## Project Overview

This project focuses on analyzing Electric Vehicle (EV) market data using Power BI and Power Query Editor. The dashboard provides valuable insights into EV market performance, sales trends, vehicle specifications, customer satisfaction, and market segmentation.

The goal of this project is to transform raw EV market data into an interactive and visually appealing dashboard that helps users understand industry trends and make data-driven decisions.

## Dataset Information

The dataset contains over 2,000 EV records with multiple attributes including:

* Brand & Model
* Vehicle Variant
* Price
* Battery Capacity
* Vehicle Range
* Charging Speed
* Horsepower & Torque
* Safety Ratings
* Customer Ratings
* Market Segment
* Country of Origin
* Annual Sales Units

## Tools & Technologies Used

* Power BI
* Power Query Editor
* DAX
* Data Cleaning & Transformation
* Data Visualization

## Data Cleaning & Preprocessing

Using Power Query Editor, the following preprocessing steps were performed:

* Handling missing values
* Removing duplicates
* Data type transformations
* Formatting columns
* Data validation
* Feature preparation for visualization

## Dashboard Features

The Power BI dashboard includes:

* KPI Cards
* Sales Trend Analysis
* EV Brand Comparison
* Battery & Range Analysis
* Customer Satisfaction Insights
* Country-wise EV Market Analysis
* Market Segment Distribution
* Interactive Filters & Slicers

## Key Insights

* Comparison of EV brands based on sales and customer ratings
* Analysis of battery capacity and driving range
* Identification of high-performing market segments
* Country-wise EV market trends
* Relationship between pricing and performance
